# 目录结构
- History.md // 项目改动历史
- INSTALL.md // 安装说明
- Makefile
- benchmark // 基准测试
- controllers // 控制器
- lib // 没有模块化的文件目录
- middlewares // 中间件
- package.json // 包描述文件，项目依赖项配置等
- proxy // 数据代理目录，类似MVC中的M
- test // 测试目录
- tools // 工具目录 
- views // 视图目录
- routes // 路由目录
- dispatch.js // 多进程管理
- README.md // 项目说明文件
- assets // 静态文件目录
- assets.json // 静态文件与CDN路径的映射文件
- bin // 可执行脚本
- config // 配置目录
- logs // 日志目录
- model
- app.js
